# Soul Code - 
<img style="display: block;margin-left: auto; margin-right: auto;" src ="/assets/images/soulcode/environment.avif" alt = "Soul Code Environment" width = "90%"  />&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Soul Code is a roguelike movement shooter early in production by <a style="color:blue;" href="https://www.gravitongames.org">Graviton Games</a>. Set in a unique sci-fi world, you traverse through a variety of space stations procedurally generated with each run while fighting enemies and upgrading yourself, both short and longterm. While being an indie project, Soul Code has a solid long-term plan and is in stable development with the goal of a commercial launch following a crowdfunding campaign.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;I have focused on creating the backbone of several core features and systems, including implementing the necessary systems for a variety of reactive effects, implementing many underlying enemy systems, and the 'Blackout' System, modifying both gameplay and visuals mid-gameplay. Rather than develop recklessly, I've made sure each piece of code is robust and serves as a long lasting foundation for the future of the project rather than accumulating tech-debt that would have to be paid later. Toward this goal, I've made sure each system is also clean, to allow for future modification and expansion, and easy and intuitive for designers to work with. I've also worked to revise, debug, and clean up previous existing code and systems to ensure it doesn't cause issue with the work built on top of it.
